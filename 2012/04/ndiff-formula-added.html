In questi giorni è stata inclusa una delle mie patch in Homebrew, la patch aggiunge ndiff (una specie di diff che ignora le eventuali piccole differenze numeriche tra file).

La cosa mi é piaciuta e servita. Facendolo ho avuto la possibilità di capire come meglio come usare Git per lo sviluppo collaborativo (con le patch e i branch per feature) e come usare anche GitHub e le “Pull request”.

L’altra cosa é che Homebrew é proprio progettato bene e studiandolo riesco a imparare molto dal suo design, sia dal modello di iterazioen con l’utente.

Al momento ho un’altra patch che ho richiesto essere inclusa. Si tratta dell’utility ren che in passato veniva distribuito con Fink. A differenza della mia pull-request per ndiff questa é ancora aperta. Mi farebbe piacere che venga accettata ma non é immediato, ci sono altre 233 richieste oltre alla mia.

Se includono o meno questo secondo pacchetto per me l’importanza è relativa.
Ho cominciato a lavorare su questi pacchetti perché servivano a mia moglie per il suo lavoro. Lei può comunque installare ren facendo riferimento al mio repository personale::

brew install https://raw.github.com/andreafrancia/homebrew/ren/Library/Formula/ren.rb

Mentre per ndiff si può semplicemente usare

brew update
brew install ndiff

Il brew udpate serve per aggiornare. Nel caso ci sia una versione vecchia di brew.
