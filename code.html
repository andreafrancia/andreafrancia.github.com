Code

  • trash-cli: (Open Source) Command line interface to the Linux trash can. Interoperates with all the desktop trash cans (KDE/Gnome/Xfce). Included in some Linux distributions. Proudly written in Python.
  • Porting of rtl8139 network interface driver, from Linux to S.Ha.R.K  (an Italian Open Source Real Time Operating System) (see http://shark.sssup.it/people.shtml). (C language)
  • Serverless shareable countdown timer: a proof-of-concept project I created for learning how to perform TDD in Javascript. It works entirely client side and nevertheless you can share your timer passing the URL via chat. It uses something like ReallySimpleHistory to persist information in the URL.
  • svn-ignore: Un piccolo script che permette di aggiungere un file all'ignore list di SVN da linea di comando (con  svn-ignore-add file-you-want-ignore). Non l'ho mai usato ma scriverlo mi ha fatto capire meglio come usare TDD in Bash.
  • micro-acquirer: utility a linea di comando per aggiornare periodicamente e automaticamente il proprio database di file RINEX (Osservazioni da ricevitori GPS). Closed source, di proprietà di Galileian Plus.
  • franciapomo: Pomodoro Timer for Windows che consuma poco schermo. Open Source. Scritto in C#. Just an hack ugly code
  • remoted-list: A library that allowed me to study deeper how RMI works but you are discouraged to use it because is a framework that hides problems. Error of Youth.
  • sisma-driver, sisma-acquirer and sisma-hci: the core components of the SISMA project. Closed source property of A.S.I.
