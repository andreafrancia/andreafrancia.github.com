Confronti numerici con ndiff

Ndiff permette di confrontare i file numericamente. E’ molto comodo se si vogliono confrontar gli output di programmi di calcolo numerico trascurando le differenze di formato di rappresentazione dei numeri.

Io lo installo cos√¨::

mkdir --parent .local/{bin,src,man/man1}

cd  .local/src
wget http://ftp.math.utah.edu/pub//misc/ndiff-2.00.tar.gz
tar xvfz ndiff-2.00.tar.gz

cd ndiff-2.00
./configure --prefix ~/.local
make install

echo 'export PATH=~/.local/bin:"$PATH"' &gt;&gt; ~/.bashrc
echo 'export MANPATH=~/.local/man:"$MANPATH"' &gt;&gt; ~/.bashrc
