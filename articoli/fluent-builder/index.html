Fluent Builder

Cosa permette di fare

Con questa libreria è possibile costruire un interfaccia un Builder Fluent per una data classe JavaBean.
La libreria permette di riscrivere il codice seguente
Person person = new Person();
person.setFirstName("John");
person.setLastName("Smith");
person.getLovedThings().add("pizza");
person.getLovedThings().add("soccer");
person.getLovedThings().add("jogging");
person.setAge(30);
nel seguente modo:
Person person2 = Person.builder()
        .withFirstName("John")
        .withLastName("Smith")
        .withLovedThings("pizza", "soccer", "jogging")
        .withAge(30)
        .build();

Utilizzo

Per aggiungere l'interfaccia di fluent building alla classe Person basta aggiungere il seguente codice alla classe Person:
import andreafrancia.util.fluent.BuilderFactory;

...

public interface Builder extends andreafrancia.util.fluent.Builder<Person> {
    Builder withFirstName(String firstName);   
    Builder withLastName(String lastName);
    Builder withLovedThings(String... lovedThings);
    Builder withAge(int age);
    Person build();
}

public static Builder builder() {
    return BuilderFactory.createBuilder(new Person(), Builder.class);
}
