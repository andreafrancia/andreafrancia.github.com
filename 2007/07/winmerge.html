WinMerge

Licenza: GPL Ambiente: Windows Confronta due file di testo. Come il comando diff sotto Unix, solo che il confronto è visuale. Come WinDiff di Visual Studio solo che funziona meglio. Supporta l'highlight syntax per un molti linguaggi. Confronta directory in modo ricorsivo. Si può configurare per ingnorare i file ausiliari di CVS e Subversion. Permette di modificare i file che si stanno confrontando. Utilissimo per lo sviluppatore. http://winmerge.org/
